# HR Analytics Employee Attrition & Performance

Projet de Machine Learning supervisé visant à prédire le risque de départ des employés à partir de données RH.

## Objectif du projet

Ce projet a pour objectif d'analyser des données de ressources humaines afin de construire et comparer plusieurs modèles de classification capables d'identifier les employés susceptibles de quitter l'entreprise.

## Pipeline du projet

1. Chargement du dataset RH
2. Analyse exploratoire des données
3. Nettoyage des données
4. Encodage des variables catégorielles
5. Séparation des données en ensembles d'entraînement et de test
6. Scaling des variables pour le modèle SVM
7. Entraînement d'un modèle Decision Tree
8. Entraînement d'un modèle Random Forest
9. Entraînement d'un modèle SVM
10. Évaluation des modèles
11. Comparaison finale des performances
12. Décision sur le modèle le plus adapté

## Modèles utilisés

- Decision Tree
- Random Forest
- Support Vector Machine (SVM)

## Résultat attendu

Le projet permet de comparer les performances des différents modèles afin de sélectionner l'approche la plus pertinente pour la prédiction de l'attrition des employés.
